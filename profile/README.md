# RenewEarth Lab

**기후 기술로 만드는 지속 가능한 공급망, 자동화된 탄소 관리를 선도합니다.**

RenewEarth Lab은 자동차 산업 공급망의 탄소 배출 데이터 수집·산정·검증·보고를
자동화하는 B2B SaaS 플랫폼 **CarbonLink**를 만듭니다.
350+개 제조 기업이 CarbonLink를 통해 CBAM, CDP, CSDDD 등 글로벌 규제에 대응하고 있습니다.

---

## CarbonLink

공급망 탄소 데이터를 자동으로 수집·산정·보고하는 올인원 솔루션

| 영역 | 설명 |
|------|------|
| **Data Collection** | 협력사 데이터 자동 수집 및 통합 관리 |
| **LCA / PCF** | 제품 탄소발자국 자동 산정 |
| **Verification** | 원격 제3자 검증 One-Click |
| **Reporting** | CBAM·CDP·CSDDD 보고서 자동 생성 |
